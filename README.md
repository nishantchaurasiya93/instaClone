# 📸 InstaClone - MERN Social Media App

A full-stack Instagram-like web application built using the **MERN stack** with **ImageKit integration** for image uploads and optimized delivery.

---

## 🚀 Features

* 🔐 User Authentication (JWT + Cookies)
* 👤 Register / Login / Logout
* 📷 Upload Posts with Images (ImageKit)
* 📝 Add Captions to Posts
* 🧑‍💻 User Profile
* 🔒 Protected Routes (Auth Middleware)
* ⚡ Fast image delivery via CDN (ImageKit)

---

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* React Router DOM
* Axios
* SCSS

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)

### Other Tools

* ImageKit (Image Upload & CDN)
* JWT (Authentication)
* bcrypt (Password Hashing)
* multer (File Upload)
